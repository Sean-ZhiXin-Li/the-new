# Phase0 Day4 — Representation System

## Core Idea

Span = reachable space from linear combinations.

Basis = minimum non-redundant set that still spans the full space.

Redundancy = vectors that do not add new reachable directions.

---

## Case A
(1,0), (0,1)
→ Span R2
→ Basis

## Case B
(1,2), (2,4)
→ Same line
→ Not Span R2

## Case C
(1,0), (0,1), (1,1)
→ Span R2
→ Redundant third vector

---

## Rank Intuition

Rank = effective independent directions.

---

## Day4 Insight

More vectors does not necessarily mean more capability.
New independent direction is what matters.
